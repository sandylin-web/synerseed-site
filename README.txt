SynerSeed final website package

Files:
- index.html
- styles.css
- script.js
- vercel.json
- images/ folder

How to update GitHub:
1. Unzip this package.
2. Open your GitHub repository: synerseed-site.
3. Upload all files and folders, replacing the existing files.
4. Commit changes.
5. Vercel will redeploy automatically.

Important:
Do not upload only the zip file to GitHub. Upload the extracted files.
