How to deploy on Vercel:
1. Go to vercel.com and sign in.
2. Click Add New -> Project.
3. Import this folder by dragging it in, or upload it from your computer.
4. Vercel will detect it as a static site. Click Deploy.
5. After deployment, open Project Settings -> Domains.
6. Add these two domains:
   - synerseed.com
   - www.synerseed.com
7. In your domain registrar DNS panel, add the exact records Vercel shows for your project.

Important:
- Vercel now recommends checking the exact DNS values in your project settings after adding the domain.
- In many cases, the apex domain uses an A record and the www subdomain uses a CNAME record.
